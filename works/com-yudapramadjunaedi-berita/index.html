Deskripsi
Berita is a news aggregator sourced from hundreds of Indonesia news sites that are summarized into 10 hottest news per topic and always update within 30 seconds

BERITA
+ Latest and hottest news from hundreds of Indonesia news sites
+ News is updated within 30 seconds
+ Compiled into a 10 news per topic
+ Select topics according to your interests. Customization on Topic page
+ Topic options: National, International, Business, Technology, Entertainment, Sports, Health, and Science

READER
+ Optimized for reading
+ 4 theme options: light, night, sepia, gray
+ You can increase or decrease text size
+ You can change font
+ Save customization on theme, font type, and text size.
+ Bookmark in order to read offline
+ Organize reader list: reorder, delete, etc.

BROWSE
+ Search news
+ Per search will produce 100 news
+ Bookmarks search results so that you can access it offline on Browse page
+ Organize browse list: reorder, delete, etc.
+ Organize the list of search results: reorder, delete, etc.

WEBSITE
+ Read news from the source using the in-app browser
+ Bookmark website so you can easily access it in the future. All website bookmark stored on Website page
+ Organize website list: reorder, delete, etc.

DISCUSSION
+ You can give comment on news
+ Bookmark discussion so that you can access it offline. Bookmarks are stored on Diskusi page
+ Organize discussion list: reorder, delete, etc.
